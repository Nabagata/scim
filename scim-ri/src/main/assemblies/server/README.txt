This package contains the UnboundID SCIM SDK Reference Implementation Server.
It contains the following elements:

  * config -- Directory containing the resource mapping XML and XSD files. 
  
  * ldif   -- Directory containing some example LDIF for use in the SCIM server.
    
  * lib    -- Directory containing SCIM server libraries and dependencies.

  * tools  -- Directory containing the SCIM server startup script and tools.
