This package contains the UnboundID SCIM Reference Implementation for Java.  It
contains the following elements:

- unboundid-scimri.jar -- The UnboundID SCIM Reference Implementation for Java
  library.

- docs/index.html -- A list of available documentation.

- examples -- A directory containing source code for example programs that
  illustrate the use of the SCIM Reference Implementation.

- LICENSE.txt -- Provides information about the licenses under which the
  UnboundID SCIM Reference Implementation for Java is made available.
